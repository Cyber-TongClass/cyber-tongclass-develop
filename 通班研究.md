## 事实（来源：百度百科 + 2023招生简章 + 北大招生办公众号 + 通班生存指南/学生组织介绍 2025-08-05）


### ⭐ 权威定性（百度百科）

| 项 | 内容 |
|---|------|
| 官方全称 | 通用人工智能实验班（简称"通班"，英文 TongClass） |
| 性质 | 北京大学与清华大学联合创立的人工智能人才培养项目 |
| 创立时间 | **2021年**（2021年1月北大启动招生，2021年5月清华启动招生） |
| 领衔创始人 | 朱松纯（北京大学讲席教授、清华大学讲席教授、北京通用人工智能研究院院长） |
| 北大归属 | 北京大学元培学院 |
| 清华归属 | 清华大学自动化系（"通用人工智能"因材施教计划） |
| 培养平台 | 依托北京通用人工智能研究院（通研院），构建本硕博贯通培养体系 |

### 培养理念（百度百科统一表述）

| 理念 | 含义 |
|------|------|
| **通识** | 了解和人工智能交叉的人文学科，结合清北优势逐步推出 AI+人文、艺术、社科、伦理等交叉课程 |
| **通智** | 掌握人工智能核心理论，涵盖视觉、语言、认知、机器人、机器学习、多智能体六大核心领域 |
| **通用** | 掌握支撑 AI 的基础学科并具有动手能力；通过 AI+行业交叉进入信息、医疗、健康、金融等赛道 |
| **最终目标** | 培养"通识+通智+通用"的 AI "通才" |

### 六大核心领域（统一口径）

计算机视觉、自然语言处理、认知推理、机器学习、机器人学、多智能体
